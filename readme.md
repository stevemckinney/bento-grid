## Setup
1. Open terminal at the location of this repo
2. Run `npm install`
3. Run `npx @tailwindcss/cli@next -i app.css -o app.output.css --watch`
4. Edit `app.css`

Optionally in a new terminal window run `python3 -m http.server` to see it.
